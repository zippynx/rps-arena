# Rock Paper Scissors

Game sederhana Rock Paper Scissors menggunakan HTML, CSS, dan JavaScript.

## Cara Main
1. Klik salah satu pilihan: Rock, Paper, atau Scissors.
2. Komputer akan memilih secara acak.
3. Skor akan bertambah sesuai hasil permainan.

## File
- `index.html` → Struktur halaman.
- `style.css` → Tampilan game.
- `script.js` → Logika permainan.

## Assets
Pastikan tersedia file gambar:
- `rock.png`
- `paper.png`
- `scissors.png`
